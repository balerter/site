```
name: ch1
host: domain.com
port: 6440
username: username
password: password
database: database
sslCertPath: /path/to/cert.crt
timeout: 3s
```

### name 

> Обязательное поле, Уникальное значение среди источников данных этого типа

Название источника данных

В названии могут использоваться латинские символы, цифры и знак подчеркивания

По этому имени вы будете обращаться из скрипта, например:

```
local ds = require('datasource.clickhouse.ch1')
```

### host

> Обязательное поле

Хост для подключения к БД

### port

> По-умолчанию `6440`

Порт для подключения к БД

### username

> По-умолчанию `default`

Имя пользователя для подключения к БД

### password

> По-умолчанию пустая строка

Пароль для подключения к БД

### database

> По-умолчанию `default`

Имя БД для подключения

### sslCertPath

> По-умолчанию пустая строка

Пусть к SSL сертификату для подключения к БД. Если пустая строка, SSL для подключения не используется

### timeout

> По-умолчанию: 5s

Таймаут
