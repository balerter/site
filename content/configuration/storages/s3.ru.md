Настройки хранилища s3

```
name: dev
region: us-east1
key: SOME_KEY
secret: SOME_SECRET
endpoint: SOME_ENDPOINT
bucket: SOME_BUCKET
```

### `name`

> Обязательное поле, Уникальное значение среди хранилищ этого типа

Название хранилища


### `region`

> Обязательное поле

Регион

### `key`

> Обязательное поле

Ключ доступа

### `secret`

> Обязательное поле

Секретный ключ

### `endpoint`

> Обязательное поле

Эндпоинт (без схемы)

### `bucket`

> Обязательное поле

Имя бакета
