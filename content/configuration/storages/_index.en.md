In the `storages` section describes settings for remote storages.

- [s3](s3)

```
storages:
  s3:
    - name: dev
      region: us-east1
      key: SOME_KEY
      secret: SOME_SECRET
      endpoint: SOME_ENDPOINT
      bucket: SOME_BUCKET
```
