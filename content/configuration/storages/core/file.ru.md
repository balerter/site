Настройки хранилища File

```
name: primaryFile
path: /path/to/file
timeout: 1s
```

### `name`

> Обязательное поле, Уникальное значение среди хранилищ этого типа

Название хранилища

### `path`

> Обязательное поле

Путь к файлу

### `timeout`

> По умолчанию: 1s

Таймаут для операций
