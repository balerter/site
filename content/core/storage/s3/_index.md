Модуль `storage.s3` позволяет загружать файлы во внешнее хранилище, подключенное по протоколу Amazon S3

### Методы

#### `uploadPNG(data[, <FILENAME>]) imageURL, error`

Загрузить изображение в формате `PNG` и получить ссылку на изображение.

При ошибке, она будет возвращена вторым параметром.

Если имя файла не указано, то оно будет сгенерировано автоматически.
Вы можете указать имя файла без расширения, оно будет добавлено автоматически.
Если вы укажете расширение `.png`, `.jpg` или `.jpeg`, оно будет отброшено

Пример:

```
local storage = require('storage.s3.dev')

local data = 'SOME_IMAGE_DATA_FROM_MODULE_CHART'

local link, err = storage.uploadPNG(data)

-- в переменной link при успешной загрузке вы получите публичную ссылку на изображение
```

### Публичная ссылка

Публичная ссылка для изображения формируется следующим образом:

```
https://<BUCKET>.<ENDPOINT>/<FILENAME>
```
