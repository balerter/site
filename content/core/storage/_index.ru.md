Модуль `storage` позволяет вам загрузить файл во внешнее хранилище, описанное в конфигурации

Подключение:

```
local storage = require('storage.<TYPE>.<NAME>')
```

где
- type - это тип хранилища
    - s3
- name - это имя хранилища, определенное в файле конфигурации


Примеры:

Часть файла конфигурации
```
storages:
  s3:
    - name: dev
      ...
```

Подключение модулей в скрипте
```
local s3 = require('storage.s3.dev')
```
