1. В клиенте Telegram ищем бота `@BotFather`

2. Командой `/newbot` создаем нового бота, указав имя и ник бота

3. Копируем полученный Access Token в конфигурацию Balerter

4. Далее требуется узнать `chat id`, куда мы собираемся отправлять сообщения. Для этого можно воспользоваться утилитой `tgtool`, входящей в пакет Balerter

```
tgtool -token <TELEGRAM_BOT_TOKEN>
``` 

или

```
docker run --rm balerter/tgtool -token <TELEGRAM_BOT_TOKEN>
```

Отправьте сообщение вашему боту и в выводе tgtool вы увидите chat id.

Так же можено добавить этого бота в какую-либо группу и написать туда. 

После этого chat id так же вписываете в файл конфигурации

5. Готово. Вы великолепны!
